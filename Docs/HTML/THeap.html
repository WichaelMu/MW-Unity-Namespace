THeap


The implementation of a Minimum or Maximum Heap.

Count


The number of elements in the heap.

CONSTRUCTOR (UInt32 MaxSize)


Summary:

Generates a new Heap, initialised with uMaxSize.

Add (T Item)


Summary:

Adds an item to this Heap.

Params:

Item: The item to add.

RemoveFirst ()


Summary:

Remove the element at the root of this Heap.

Returns:

The element that was removed.

UpdateItem (T Item)


Summary:

Updates Item's position in the Heap.

Params:

Item: The item to update.

UpdateItemUp (T Item)


Summary:

Sorts this Item upawrds.

Params:

Item: The item to update.

UpdateItemDown (T Item)


Summary:

Sorts this Item downwards.

Params:

Item: The item to update.

Contains (T Item)


Summary:

Whether or not this Heap contains Item.

Params:

Item: The Item to check.

Returns:

True if Item exists in this Heap.

Array ()


Summary:

The Heap as a T[].

Returns:

T[] in the order of this THeap.
