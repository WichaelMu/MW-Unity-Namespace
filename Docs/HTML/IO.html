I


Mouse Input and Keyboard Input, based off of Input.

Click (EButton BMouse, Boolean bHold, Boolean bUp)


The EButton press to listen for.

BMouse: The EButton press to listen for.

bHold: Whether or not to check if this button is held down.

bUp: Whether or not to check if this button is released.

Key (KeyCode KCStroke, Boolean bHold, Boolean bUp)


The KeyCode that was pressed on the keyboard.

KCStroke: The KeyCode that was pressed on the keyboard.

bHold: Whether or not to check if this button is held down.

bUp: Whether or not to check if this button is released.

Any ()


Identical to Input.anyKey.

Out (Object[] debug)


Identical to Log.P(object[]).

debug: The list of objects to log separated by a space.
