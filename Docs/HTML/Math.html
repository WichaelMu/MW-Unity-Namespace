public static void

Params:

        The Sine result over AngleInDegrees.

Cosine:

        The Cosine result over AngleInDegrees.

Angle:

        The angle in degrees.

bUseExactValue:

        True to use MathF.Sin and MathF.Cos for the exact SinCos values.
