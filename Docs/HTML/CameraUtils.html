public class : MonoBehaviour

SpringArm


A Spring Arm Component that allows a Boom to track a Target.



public Transform

Boom

The Transform that will track Boom.



public Transform

Target

The Target for Boom to track.



[SerializeField] Vector3

TargetOffset

The offset at which Boom will track Target.



public float

Distance

The distance in which Boom will maintain from Target whilst tracking.

Will not be used when the line between Boom and Target is interrupted by a collision.



public protected MRotator

GimbalRotation

The world-rotation of Boom, about Target + TargetOffset.



public protected MRotator

CameraRotation

The world-rotation of Boom as it looks and faces Target.



[SerializeField] bool

bInheritPitch

True to inherit the Target's Pitch rotation.



[SerializeField] bool

bInheritYaw

True to inherit the Target's Yaw rotation.



[SerializeField] bool

bInheritRoll

True to inherit the Target's Roll rotation.



public bool

bEnableScrollToDistance

True to let the mouse wheel control Distance.



public float

ScrollSensitivity

How much will Distance change when scrolling?



[SerializeField] Vector2

MinMaxDistance

The minimum and maximum Distance of the Spring Arm when Scrolling.



[SerializeField] float

OrbitSensitivity

How much will the Spring Arm rotate around Target?



[SerializeField] Vector2

MinMaxOrbitAngle

The minimum and maximum angle of the Spring Arm when Orbiting.



[SerializeField] bool

bPermanentlyChangePanTargetOffset

True to permanently change the TargetOffset when panning.



[SerializeField] float

MaxPanDistance

The maximum distance of the Spring Arm's offset to Target when Panning.



public bool

bInvertX

True to invert the Controls when Orbiting around the world X-Axis.



public bool

bInvertY

True to invert the Controls when Orbiting around the world Y-Axis.



public bool

bInvertZ

True to invert the Zooming Distance direction.



[SerializeField] bool

bRunCollisionChecks

Should this Spring Arm run Collision Checks against OnlyCollideWith?



[SerializeField] bool

bUseAdvancedCollisionBehaviour

Use Advanced Collision Behaviour upon Collision?

This automatically rotates and swivels the camera out of the way when the Target has backed up against a wall of OnlyCollideWith.



[SerializeField] bool

bForceAdvancedBehaviour

Should this Spring Arm force Advanced Collision Behaviour?

Advanced Collision Behaviour will run, regardless of if Target is backed up against a wall of OnlyCollideWith.



[SerializeField] float

AdvancedCollisionActivationDistance

The Distance in which to activate Advanced Collision Behaviour.



[SerializeField] float

MaxAdvancedCollisionAngle

The maximum angle the Advanced Collision Behaviour will rotate and swivel.



[SerializeField] LayerMask

OnlyCollideWith

Interrupt Distance when a collision is made with this layer/s.



[SerializeField] float

PositionalLagStrength

The strength Boom will Lag behind tracking Target.

0 is standstill (infinite Lag). 1 is instant (no Lag).



[SerializeField] float

RotationalLagStrength

The strength Boom will rotate to face Target.

0 is standstill (infinite Lag). 1 is instant (no Lag).
