public static class

Fast


Fast implementations of common Mathematical functions using Bit Magic.



public static unsafe float

InverseSqrt (float N, int Iterations)

Summary:

1 / sqrt(N).

Remarks:

Modified from: The 'Quake III Fast Inv. Sqrt Algorithm'

Params:

N:

        1 / sqrt(x) where x is N.

Iterations:

        The number of Newton Iterations to perform.

Returns:

An approximation for calculating: 1 / sqrt(N).
