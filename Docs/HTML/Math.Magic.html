Fast

Fast implementations of common Mathematical functions using Bit Magic.

InverseSqrt (float N, int Iterations)
Summary:
1 / sqrt(N).
Remarks:
Modified from: The 'Quake III Fast Inv. Sqrt Algorithm'
Params:
N: 1 / sqrt(x) where x is N.
Iterations: The number of Newton Iterations to perform.
Returns:
An approximation for calculating: 1 / sqrt(N).

Sqrt (float F, int Iterations)
Summary:
Faster version of Mathf.Sqrt().
Iterations: The number of Newton Iterations to perform.
Returns:
An approximation for the Square Root of F.
