Log

Write to the console.

P (Object[] args)
Summary:
UnityEngine.Debug.Log(object) every object with object.ToString().
Params:
args: The list of objects to log separated by a space.

E (Object[] args)
Summary:
UnityEngine.Debug.LogError(object) every object with object.ToString().
Params:
args: The list of objects to log separated by a space.

W (Object[] args)
Summary:
UnityEngine.Debug.LogWarning(object) every object with object.ToString().
Params:
args: The list of objects to log separated by a space.
Stacktrace

Provides a stacktrace for code.

Here (EVerbosity Verbosity)
Summary:
Stacktrace using Log.P with EVerbosity verbosity.
Params:
Verbosity: The EVerbosity verbosity of the Log.P.

Here (String Message, EVerbosity Verbosity)
Summary:
Stacktrace using Log.P with EVerbosity verbosity and a string message.
Params:
Message: The string message to show.
Verbosity: The EVerbosity of the Log.P.
Stopwatch

Track execution time of code.

CONSTRUCTOR (bool bStartOnCreation)
Summary:
Creates a new Stopwatch.
Params:
bStartOnCreation: Immediately starting timing upon instantiating?

Start ()
Summary:
Start Stopwatch.

Stop ()
Summary:
Stop Stopwatch and get the elapsed Time.
Returns:
The elapsed time before stopping.

Restart ()
Summary:
Restarts Stopwatch and get the Time before restarting.
Returns:
The elapsed time before restarting.

Time ()
Summary:
Get the current elapsed time.

TimeInSeconds ()
Summary:
Get the current elapsed time in seconds.

ToSeconds (long Milliseconds)
Summary:
Converts milliseconds to seconds.
Params:
Milliseconds: Milliseconds to convert.
