MAudio


The Audio controller for in-game sounds.

AudioInstance


A unique reference to the only Audio object in the scene.

Sounds


Every sound that this Audio object will control.

Initialise (MSound[] Sounds)


Summary:

Populates the Sounds array to match the settings.

Remarks:

If a duplicate name is detected, a failure will occur. In this case, check the names of the MSounds array.

Params:

Sounds: The sounds to initialise into the game.

Play (String Name, bool bOverlapSound)


Summary:

Plays sound of name n.

Params:

Name: The name of the requested sound to play.

Stop (String Name)


Summary:

Stops sound of name n.

Params:

Name: The name of the requested sound to stop playing.

StopAll ()


Summary:

Stop every sound in the game.

Find (String Name)


Summary:

Returns a sound in the Sounds array.

Params:

Name: The name of the requested sound.

Returns:

The MSound of the requested sound. Null if Name could not be found.

IsPlaying (String Name)


Summary:

Whether or not sound of name n is playing.

Params:

Name: The name of the sound to query.

Source


The source of this sound.

Name

Volume


Sound settings.

Pitch


Sound settings.

bLoop


Should this sound loop?

bPlayOnAwake


Should this audio clip automatically play on awake?

bMute


Should this audio clip mute?



Visualiser

Samples


The number of samples in the audio clip.

Analyse (AudioSource Source, MVector[]& Visualiser, EComponentAxis PositiveVisualiserAxis)


Summary:

Visualises sound emitted by an AudioSource.

Params:

Source: The source of the sound.

Visualiser: The sound visualiser displayed as an array of MVectors.

PositiveVisualiserAxis: How to visualise the sound in Visualiser.

Returns:

VisualInformation. Notably, decibels and pitch of the analysed sound.
