Line


Drawing LineRenderers in the game.

DrawLine (LineRenderer LRSelf, Vector vFrom, Vector vTo, Color colStartColour, Color colEndColour, float fLineWidth, Vector vOffset, Material MMaterial, Boolean bUseWorldSpace)


Draws a line from to to in StartColor to EndColor at LineWidth thickness with an offset at UseWorldSpace with NumberOfSegments.

LRSelf: The LineRenderer of the GameObject calling this.

vFrom: The coordinates where the line will originate.

vTo: The coordinates where the line will end.

colStartColour: The starting colour gradient for this line.

colEndColour: The ending colour gradient for this line.

fLineWidth: The thickness of this line.

vOffset: The offset to place this line.

MMaterial: The material used to draw the line.

bUseWorldSpace: Should this line use world space?

DrawCircle (LineRenderer LRSelf, Vector vAround, float fRadius, Color colLineColour, float fLineWidth, Material MMaterial, Boolean bUseWorldSpace, Int2 nNumberOfSegments)


Draws a circle with a centre at around at radius with a LineColour at LineWidth thickness at UseWorldSpace with NumberOfSegments.

LRSelf: The LineRenderer of the GameObject calling this.

vAround: The centre of the circle to be drawn.

fRadius: The radius of this circle.

colLineColour: The colour of this circle.

MMaterial: The material used to draw the circle.

bUseWorldSpace: Should this circle use world space?

nNumberOfSegments: The number of verticies of this circle.

World space Canvas scaling and TextMeshProUGUI utilities.

ETypewriterMode


Append to the current TextMeshProUGUI.

ETypewriterMode


Clear the current TextMeshProUGUI before typewriting.

ScaleSize (Vector vSelf, Vector vScaleWith)


Scales the canvas element relative to self.

vSelf: The position to scale from.

vScaleWith: The position to scale with.

Typewrite (TextMeshProUGUI TMPro, MonoBehaviour Game, String Content, float Delay, ETypewriterMode EMode)


Animates a TextMeshProUGUI to display Content like a typewriter.

TMPro: The extended TextMeshProUGUI GameObject.

Game: The MonoBehaviour that will be responsible for invoking the Typewriter coroutine.

Content: The content to typewrite.

Delay: The time gap between writing a new character.

EMode: Should the text ETypewriterMode.Append, or ETypewriterMode.Clear?

Typewrite (TextMeshProUGUI TMPro, MonoBehaviour Game, String Content, float Delay, ETypewriterMode EMode, String Sound, Boolean bOverlapSound)


Sound: The MSound in MAudio._AudioInstance to play when writing a character.
